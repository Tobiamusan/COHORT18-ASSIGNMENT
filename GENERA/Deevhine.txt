git init: Initializes a new local Git repository.git clone : Copies an existing remote repository.git config --global user.name "": Sets your commit author name.git config --global user.email "": Sets your commit email address.git config --list: Displays all current configuration settings.git status: Lists modified, staged, and untracked files.git diff: Shows line-by-line differences for unstaged changes.git diff --staged: Shows line-by-line differences for staged changes.git add : Adds a specific file to the staging area.git add .: Adds all modified and untracked files to staging.git commit -m "": Saves staged changes to history with a message.git commit --amend: Modifies the very last commit.git log: Displays the history of commits.git log --oneline: Condenses commit history into single lines.git show <commit_hash>: Shows details and file changes for a specific commit.git blame : Displays who changed each line of a file and when.git branch: Lists all local branches.git branch : Creates a new branch.git switch : Changes your working directory to another branch.git checkout -b : Creates a new branch and switches to it.git merge : Joins a specified branch into your active branch.git branch -d : Deletes a local branch.git remote add  : Links your local repository to a remote server.git remote -v: Lists all saved connections to remote servers.git fetch: Downloads historical data and updates from a remote server.git pull: Downloads remote changes and merges them into your active branch.git push  : Uploads local commits to a remote repository.git restore : Discards unstaged changes in your working directory.git reset HEAD : Removes a file from the staging area.git reset --soft <commit_hash>: Moves branch pointer backward while keeping changes staged.git reset --hard <commit_hash>: Permanently wipes out all changes up to a specific commit.git revert <commit_hash>: Creates a new commit that rolls back an old commit.git stash: Temporarily shelves uncommitted changes.git stash pop: Restores the most recently stashed changes.git stash list: Displays all items currently savede in your stash
